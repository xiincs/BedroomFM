# 产品名称

- BedRoom FM

------

技术栈：Go + Vue3 + MySQL + WebSocket + 网易云API

# 一、核心玩法

## 房间制

创建房间：

```
505宿舍音乐房
邀请码：A7KD9
```

成员进入后看到：

- 当前播放歌曲（歌词/封面等）
- 点歌队列
- 在线成员
- 聊天区

类似：

```
正在播放

周杰伦 - 晴天

进度：
01:23 / 04:30
```

------

## 点歌

用户搜索网易云歌曲：

```
搜索：
晴天
稻香
夜曲
```

点击：

```
+ 加入队列
```

队列：

```
1. 晴天（小王）
2. 稻香（老张）
3. 夜曲（阿豪）
```

------

## 轮流点歌机制

避免一个人霸榜。

规则：

```
A点一首
B点一首
C点一首

然后再轮回
```

排序：

```
A1
B1
C1
A2
B2
```

体验会非常好。

------

# 二、互动设计（重点）

## 1. 顶歌

类似弹幕网站的投票。

房间内每人定时获得：

```
10个顶歌币
```

可以：

```
+1
+3
+5
```

例如：

```
稻香 12票
夜曲 8票
晴天 6票
```

票数高自动提前。

------

## 2. 切歌投票

有人放神曲：

```
爱情买卖
```

全宿舍崩溃。

触发：

```
发起切歌
```

超过50%：

```
自动下一首
```

------

## 3. 表情轰炸

歌曲播放时：

```
❤️
🔥
😭
🐶
```

实时飘屏。

类似：

```
❤️❤️❤️❤️❤️
🔥🔥🔥🔥
```

实现简单但互动感极强。

------

## 4. 歌曲PK（先不做）

房主发起：

```
晴天 VS 七里香
```

成员投票：

```
晴天：7票
七里香：5票
```

赢家下一首播放。

------

## 5. 猜点歌人（先不做）

歌曲播放后：

```
猜猜是谁点的？
```

选项：

- 小王
- 阿豪
- 老张

猜中：

```
+10积分
```

特别适合宿舍。

------

# 三、轻社交系统

## 音乐人格

根据点歌自动生成。

例如：

```
emo达人
周杰伦狂热粉
二次元DJ
粤语老歌守护者
```

显示在头像下：

```
阿豪
[深夜emo王]
```

------

## 今日MVP

每天统计：

- 点歌数
- 获赞数
- 顶歌数

产生：

```
今日DJ 👑
```

------

# 四、积分系统

获得积分：

| 行为       | 积分 |
| ---------- | ---- |
| 点歌       | +1   |
| 顶歌       | +1   |
| 猜中点歌人 | +10  |
| 歌曲获赞   | +3   |

兑换：

- 专属头像框
- DJ称号
- 彩色昵称
- 房间特效
